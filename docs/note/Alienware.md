## eGPU

> EGPU 是“外部图形处理单元”(ExternalGraphics Processing Unit)的简称(你猜到了)，它是一个多年来一直在互联网上流传的想法，而且在显卡制造商的研发部门中的时间更长了。在不涉及太多技术的情况下，一般的概念是通过一条电缆将一台普通笔记本电脑连接到外部显卡上，这样就可以从笔记本电脑的内部 GPU 中卸下所有的负载，并将其放到功能更强大的扩展上。
