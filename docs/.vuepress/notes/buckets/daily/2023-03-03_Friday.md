#pnpm 

如果修改了 `node_modules` 文件，使用 pnpm 安装的项目依赖包会被全部影响。

例如，项目 A 使用了依赖包 foo，如果无意间修改了依赖包 foo 中的内容，那么其他项目中的引用的 foo 也会被修改。

解决办法：

```sh
# 进入项目中，删除 foo
pnpm remove foo
# 从 store 中删除 foo 
pnpm prune foo
# 再次安装 foo
pnpm add foo
```

[pnpm prune](https://pnpm.io/zh/cli/prune)
