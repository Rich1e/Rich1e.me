## git 找回代码

恢复特定 commit 中的某个文件

```git
git checkout <目标 commit> -- <文件>
```

文件回滚

```git
# 仅在工作区修改，还没有提交暂存区和本地仓库
git checkout -- 文件名
# 添加到暂存区，但还未提交 commit
git reset HEAD 文件名
# 已提交commit，但还没有 push 时
git reset <要回滚到的 commit>
# 已 push 到远端时
git revert
```
