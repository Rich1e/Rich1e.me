## 技术方案

mathjs 二次扩展

[math.js | an extensive math library for JavaScript and Node.js](https://mathjs.org/)

## 产出内容

npm package

## 业务场景

- GUI - 单机版
- SDK - 服务端

## 使用功能

- 表达式计算
- 表达式校验
- 输入联想

## API 设计

```js
import GlobalEvaluator from 'global-evaluator'

const geval = new GlobalEvaluator();

// 表达式计算
geval.computed();

// 表达式校验
geval.checked();

// 输入联动
geval.autoComplete();
```
